# Operational Bottlenecks & Root Cause Analysis

This project analyzes operational workflow data to identify bottlenecks, root causes of delays, SLA breaches, and process improvement opportunities.

## Tools Used
- Python
- SQL / PostgreSQL
- Power BI
- GitHub
- VS Code

## Project Goals
- Generate and clean operational process data
- Analyze stage-level bottlenecks
- Identify root causes of delay
- Measure SLA breach percentage
- Build Power BI dashboard
- Create hiring-ready data analytics portfolio project

## Project Structure
- data/raw: original datasets
- data/processed: cleaned datasets
- sql: SQL scripts and views
- notebooks: Python analysis
- powerbi: dashboard files
- reports: business report and summary
