# 🚀 Operational Bottlenecks & Root Cause Analysis

## 📌 Project Overview
This project analyzes operational workflow data to identify bottlenecks, measure delays, and uncover root causes using SQL, Python, and Power BI.

It simulates a real-world analytics pipeline, combining data engineering, analysis, and machine learning to provide actionable business insights.

---

## 🎯 Objectives
- Identify bottleneck stages in operations
- Analyze delay patterns and SLA breaches
- Perform root cause analysis
- Predict late orders using machine learning
- Provide data-driven business recommendations

---

## 🛠 Tech Stack
- **SQL (PostgreSQL)** – Data storage, transformation, and KPI calculation
- **Python (Pandas, Scikit-learn)** – Data analysis and machine learning
- **Power BI** – Dashboard and visualization
- **GitHub** – Version control and project showcase
